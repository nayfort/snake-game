# Snake Game

This is a simple Snake game built with Pixi.js and Vite.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/snake-game.git
   cd snake-game

2. Install the dependencies:
   ```bash
   npm install

3. Run the development server:
   ```bash
   npm run dev

4. Build:
   ```bash
   npm run build
